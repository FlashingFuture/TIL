### 자주 활용되는 파이썬 패키지
#### 데이터 처리 / 데이터 분석
Numpy
- 배열에 최적화된 수학 계산용 패키지
    - 행렬 연산이 Python 반복문보다 빠름
    - 다차원 행렬 자료구조, 행렬 인덱싱 기능 제공
Pandas
- 원하는 데이터만 추출하거나 분석할 때 사용되는 패키지
    - Numpy를 기반으로 그룹화 / 피버 등의 구조화를 쉽게 할 수 있음
#### 데이터 시각화
Matplotlib
